## 步骤①: 启动前端服务

> - 首先安装vue-cli脚手架，直接在任意目录执行`npm install vue-cil@3 -g `对应版本为3系列
> - 然后使用`vue -V`查看版本

![](img/05.png)

`clone`或`download`项目后, 直接在对应的目录执行`npm install`, 直接把前端服务所需要的依赖全部安装, 然后再执行对应的`npm run serve`即可启动前端服务

![](img/03.png)
